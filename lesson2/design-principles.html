Repetition

DND Beyond

dndbeyond.com

I chose this website for the repetitive theme it uses to make everything appear organized and concise with all the items the webpage needs to display. The first four icons have the same layout and look, then as we scroll down we can see the rest of the objects that are repeted as well.

Visual Hierarchy

GitHub

github.com

GitHub does a good job displaying the most important items in larger frames to catch the user's eye when they browse the page. This really helps anyone who is looking for something specific that is used frequently on that webpage.

Constrast

VidAngel

vidangel.com

In this website we see that the overall tone is dark in contrast to the movies and TV shows that they have on for display. This not only enchances the user's search for their particular show, but also gives it a good appeal overall.
